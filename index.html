Windninja

WindNinja is a diagnostic wind model developed for use in wildland fire modeling.

Building on Windows

Directories:

  • autotest -> testing suite
  • cmake -> cmake support scripts
  • data -> testing data
  • doc -> documentation
  • images -> splash image and icons for gui
  • src -> source files

Dependencies (versions are versions we build against):

  • Boost 1.46:
    • boost_date_time
    • boost_program_options
    • boost_test
  • NetCDF 4.1.1
  • GDAL 1.11.1
    • NetCDF support
    • PROJ.4 support
    • GEOS support
    • CURL support
  • Qt 4.7.4
    • QtGui
    • QtCore
    • QtNetwork/Phonon
    • QtWebKit
  • OpenFOAM 2.2.x
